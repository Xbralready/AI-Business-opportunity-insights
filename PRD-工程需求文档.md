## 1. 产品概述

### 1.1 产品定位

AI 信贷员是一款面向银行对公客户经理的智能销售情报系统，通过 AI 技术为客户经理提供客户洞察、产品推荐、行动指导等全流程销售支持。

### 1.2 目标用户

- **主要用户**：银行对公客户经理
- **用户场景**：日常客户拜访、电话沟通、业绩管理、客户分析

### 1.3 核心价值

| 价值点 | 描述 |
|-------|------|
| AI 客户洞察 | 多维度智能分析客户画像，自动识别商机 |
| 产品智能推荐 | 基于客户需求自动匹配最优产品组合 |
| 行动指导 | 提供具体可执行的销售行动建议和话术 |
| 业绩可视化 | 实时展示业绩数据和目标完成度 |

### 1.4 技术架构

| 技术栈 | 版本 | 用途 |
|-------|------|-----|
| React | 19.2.0 | 前端框架 |
| TypeScript | 5.9.3 | 类型安全 |
| Vite | 7.2.4 | 构建工具 |
| React Router | 7.10.1 | 路由管理 |
| Framer Motion | 12.23.25 | 动画效果 |
| Lucide React | 0.556.0 | 图标库 |
| TailwindCSS | 内置 | 样式框架 |

**筛选逻辑**：

```
filteredCustomers = mockCustomers.filter(customer => {
  // 1. 只显示对公客户
  if (customer.customerType !== 'corporate') return false;

  // 2. 搜索词匹配（客户名称/行业/AI洞察/需求标签）
  if (searchTerm) {
    const searchLower = searchTerm.toLowerCase();
    const matchName = customer.companyName.toLowerCase().includes(searchLower);
    const matchIndustry = customer.industry.toLowerCase().includes(searchLower);
    const matchInsight = customer.aiInsight?.toLowerCase().includes(searchLower);
    const matchTags = customer.needTags?.some(tag =>
      tag.toLowerCase().includes(searchLower)
    );
    if (!matchName && !matchIndustry && !matchInsight && !matchTags) return false;
  }

  // 3. 客户类型筛选
  if (customerLevelFilter && customer.customerLevel !== customerLevelFilter) {
    return false;
  }

  // 4. 行业筛选
  if (industryFilter && customer.industry !== industryFilter) {
    return false;
  }

  return true;
});
```
